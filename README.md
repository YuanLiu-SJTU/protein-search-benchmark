## Project Structure and Tasks

This repository contains benchmarking tasks for evaluating protein remote homolog search methods.
Each task folder contains all necessary scripts and input files to reproduce the benchmark results shown in the paper. Users can run the scripts independently per task to evaluate a method’s performance.
